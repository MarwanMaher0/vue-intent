### Added
- Initial implementation of vue-intent package
- `useIntent` composable for consuming intent state
- `useIntentNavigation` composable for navigation protection
- `useIntentState` composable for granular state access
- `IntentGuard` component for permission-based rendering
- `IntentMessage` component for displaying intent messages
- `IntentProgress` component for multi-step workflows
- `v-intent` directive for binding intent state to DOM elements
- `VueIntentPlugin` for global registration
- Full TypeScript support with type definitions
- Comprehensive test suite with Vitest
- Documentation and examples
- Basic example application
