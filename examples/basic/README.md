# Basic Example

Runs a simple demo app showing:
- `IntentGuard`, `IntentMessage`, `IntentProgress`
- `v-intent` directive
- `useIntent`, `useIntentState`, `useIntentNavigation`

## Run

From the repo root:

```bash
npm install
npm run dev
```

Vite will start the example app (configured via `vite.example.config.ts`).
